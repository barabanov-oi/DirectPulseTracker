Token Information
About Yandex Direct Tokens

Yandex Direct tokens are OAuth 2.0 access tokens that allow DirectPulse to access user's Yandex Direct advertising accounts.

Each token is associated with a specific user and can be used to:

  • Fetch campaign statistics
  • Generate reports
  • Monitor campaign performance

Tokens expire after a certain period and are automatically refreshed when needed.

Access Control

Deleting a token will immediately revoke DirectPulse's access to the user's Yandex Direct account.

Users will need to reconnect their Yandex Direct account to continue using automated reports and alerts.

For security reasons, full token values are never displayed or stored in plaintext.
